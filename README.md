# Izembek Fall Pacific Brant Photographic Survey

## Overview


## Installation

R v4.3.1 or higher is required. Necessary packages are documented in code files. 

For running machine learning and photo inference, Python is needed, see [usgs-geese](https://github.com/agentmorris/usgs-geese). 

## Usage

Code is presented only for documentation purposes. Download individual code files and data to a local directory, then run using R v4.3.1. 

## Getting help

Contact [Erik Osnas](mailto:erik_osnas@fws.gov) or [Charles Frost](mailto:charles_frost@fws.gov) for questions about this repository, source data, data products, or to report bugs or data errors.

Contact the project maintainers for information about contributing to this repository. Submit a [GitHub Issue](https://github.com/USFWS/Fall-Brant-Photo-Survey/issues) to report bug, request a feature or enhancement, or to report a data quality problem.
